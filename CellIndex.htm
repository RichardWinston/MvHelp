The i-j-k Index System for Cells

  • For finite-difference models (MODFLOW-2000, MODFLOW-2000 GWT, MODFLOW-96, MOC3D, and MT3DMS)
    • The index i runs along a row.
    • The index j runs along a column.
    • The index k increases from one layer to the next, counting from bottom to top. Note that this is opposite to the MODFLOW convention in which layers are counted from top to bottom.
  • For the finite-element model SUTRA
    • The index i runs along the "first numbering direction"
    • The index j runs along the "second numbering direction"
    • The index k runs along the "third numbering direction"
