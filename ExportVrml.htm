Exporting the Display as a VRML file

  1. Click the File menu and select Export As VRML.
  2. Navigate to the folder in which you want to save the VRML file.
  3. In the File Name box, enter the name of the VRML file. This name should end with the extension .wrl
  4. Click Save.

Note:

  • The VRML file may be opened and viewed in a VRML viewing application.
