New Features in Version 1.1

  • A line drawing contained in an ESRI Shapefile or a DXF file may be displayed on the top surface of the model grid or on a flat plane above the model grid.
  • Display of particle locations in MODFLOW-2000 GWT and MOC3D
  • Support the final release of SUTRA version 2D3D.1. The beta version of this model is no longer supported.
  • A number of bugs in Model Viewer 1.1 has be fixed.

Note: MV files saved by Model Viewer 1.0 can be opened in version 1.1.
