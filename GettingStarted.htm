Getting Started

  1. To use Model Viewer, you must first run a model to generate simulation results. Alternatively, you may use the example data provided with this software.
  2. Create a new MV file and load model data.
  3. Learn to zoom, rotate, and pan.
  4. View scalar data as a solid or a set of isosurfaces.
  5. View vector data, if available.
  6. View pathlines, if available.
  7. View cells or nodes that represent model features such as streams and wells.
  8. View auxiliary graphic objects such as grid lines and bounding box.
  9. Learn how to use the toolboxes for data exploration and for controlling various aspects of the display.
  10. Save the MV file.
